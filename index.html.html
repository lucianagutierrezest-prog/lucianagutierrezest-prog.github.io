Pregunta 3

Existen posibles problemas de seguridad al usar dispositivos controlados por microprocesador en el hogar.

a. Describe estos problemas y explica cómo se pueden mitigar.

(Escribe aquí tu descripción)

Respuesta:Tener un dispositivo controlado por microprocesador conectado a internet puede generar problemas de ciberseguridad, si tu puedes comunicarte remotamente con los dispositivos de tu hogar, un hacker también puede hacerlo. Cualquier dispositivo doméstico que pueda controlarse remotamente podría permitir a un hacker obtener datos personales sobre ti. Estos dispositivos suelen tener una contraseña predeterminada (o ninguna), lo que facilita a los ciberdelincuentes obtener información personal. Por ejemplo si la nevera pide automáticamente comida de un supermercado, un hacker puede obtener datos clave, como números de tarjetas de crédito.Algunas soluciones podrian ser cambiar las contraseñas predeterminadas y usar contraseñas únicas para cada dispositivo, o instalar actualizaciones de software regularmente para protegerse de vulnerabilidades.
